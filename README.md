# Monesh S — Portfolio

> 🌐 **Live Website:** [moneshs.netlify.app](https://moneshs.netlify.app/)

A professional, high-performance interactive portfolio showcasing my work as an AI Researcher and Founder. 

Built at the intersection of **Quantum Machine Learning**, **Deep Learning**, **Computer Vision**, and **Art**.

## 🚀 Tech Stack

* **Core:** React, Vite
* **Styling:** Custom Vanilla CSS (Modern Grid, Glassmorphism, Animations)
* **Hosting:** Netlify
* **Dependencies:** Framer Motion, GSAP, Recharts

## 🔬 Featured Domains

1. **Quantum Machine Learning (QML)**: Variational Quantum Circuits and 4-qubit system implementations using PennyLane.
2. **Deep Learning (LSTM & TFT)**: Hybrid models for wind forecasting (93%+ accuracy) and ISP anomaly detection.
3. **Computer Vision & Pose AI**: Real-time 33-keypoint BlazePose tracking at 60 FPS for the **MORIG AI Dance Judging System**.
4. **LLMs & RAG Pipelines**: Multi-turn conversational memory, Firecrawl URL-to-Reel pipeline, and ChromaDB integrations.

   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
